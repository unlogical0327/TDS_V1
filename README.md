# TDS_V1
Soleilware's TDS software is a pallet detection system with online pallet detection function and offline algorithm showcase. TDS is designed to work with RGB-D camera and offline function shows the performance with captured data.
TDS system is working to detect various types of pallets in warehouse and is able to give pallet's accurate center location in =/-5mm and pose angle with 0.25 degree in 2 meter distance(some pallet could be recognized in 3 meters). Also, it could give goods on the pallet and nearby rack location, depends on different CAN request commands. TDS System support CAN and Ethernet interface.
